19年面试清单（一）

19年3月28日，离开了工作3年的单位，4月11日开始了面试之旅，没想到的是今年找工作异常艰难。

普通题

  1. echo count(strlen(“http://php.net"))
    遇见次数：2
    手册解释：count传入非数组值会返回1，如果传入的值为NULL，返回0
    php5.4 输出 1
    php7.3 将会返回一个warning级别的错误，Parameter must be an array or an object that implements Countable.

  2. PHP 加密方式
    md5() crypt() sha1() urlencode() base64_encode()

  3. 不使用第三个变量交换两个变量
    出现次数： 1
    方法一： 使用异或操作。
    方法二： 使用list($a, $b) = array($a, $b), 注意php7 会改变$a, $b 顺序。
    方法三： 字符串可以将两变量拼接后去截取。

  4. shell 获取文件每行第三个元素 awk ‘{print $3}’ test”

  5. shell 输出当前的pid
    $$: 当前shell的pid
    $!：上一个进程的pid
    $?: 上个命令推出状态或函数返回值
    $0: 当前进程名
    $n: 传递给脚本第n参数
    $#: 传递给脚本或本函数的参数个数
    $*: 传给脚本的所有参

程序题
  1. 遍历文件夹
    出现次数：1
  1. 多进程同时读写一个文件
    协同锁、LOCK_SH、LOCK_EX、LOCK_UN、LOCK_NB、
  1. 冒泡排序
  2. 最近最少使用缓存失效策略算法实现 LRU

  3. 内存256M， 10G文件排序实现 file sort

    数据库题

  4. mysql快速清空数据库

  5. mysql 查学生各科分数前三
    student_score{id,name,subject_id,score}

  6. 数据库评论回复功能 详见CSDN某大神著作《数据库设计——评论回复功能》

那些打不死我的，必将使我更加强大。
