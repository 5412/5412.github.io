mac系统通过docker搭建kafka环境

发表于 2022-04-14

背景

本地开发任务需要启动个 kafka 环境, 模拟测试

环境

  1. MacOS
  2. Docker Desktop Version 4.6.1
  3. HOST_IP=宿主机网络IP | 个别情况可以使用 host.docker.internal
  4. KAFKA_TOPIC=xxxxxxx
  5. KAFKA_GROUP=xxxxxxx

安装步骤

  1. 启动 zookeeper

  2. 启动 kafka

  3. 创建 topic

  4. 创建 consumer

  5. 创建 producer

  6. 查看 group 状态
